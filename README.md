# Road Crashes Victoria

## Dependencies
**Python**
- Version 2.7

*The webapp not guaranteed to work using other version of Python*

**Python Libraries** 
- Flask
- pandas
- numpy

*If Anaconda Python distribution is being used, then no Python libraries need to be installed 
Otherwise all the listed libraries can be installed using `pip` from the command line*

**Browser**
- JavaScript enabled
- Support for the latest HTML5 and CSS3 standards

*The webapp is not guaranteed to work on any browser that does not meet these requirements*

## Running the Webapp
1. Extract the files from the zip file
2. From the command line, run all Python scripts in the “python/pre-processing” folder **(NOTE: This has already been done)**
3. From the command line, run the `app.py` file, making sure Flask is installed.
4. Navigate to http://127.0.0.1:80 or http://localhost in a browser to view the webapp
